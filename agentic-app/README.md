## MobileWatcher — آنالیز قیمت موبایل

داشبورد وب که هسته تحلیلی ربات تلگرام برای مقایسه قیمت موبایل را نمایش می‌دهد. پروژه با Next.js 14 (App Router) و Tailwind CSS ساخته شده و برای استقرار روی Vercel آماده است.

### امکانات

- لیست به‌روز از گوشی‌های پرطرفدار با چندین قیمت از فروشگاه‌های معتبر
- مقایسه بلادرنگ با مرتب‌سازی بر اساس پایین‌ترین قیمت، اختلاف بازار، برند و تاریخ عرضه
- فیلتر برند، بازه قیمتی و جستجو روی ویژگی‌ها
- نمای خلاصه بازار شامل میانگین قیمت، اختلاف میانه و بهترین پیشنهاد روز
- وبهوک آماده `/api/bot` برای اتصال مستقیم به ربات تلگرام (با پشتیبانی از دستورات `/start`, `/market`, `/top` و جستجوی متنی)

### اجرای محلی

```bash
npm install
npm run dev
```

سپس آدرس [http://localhost:3000](http://localhost:3000) را باز کنید.

### پیکربندی ربات تلگرام

برای فعال کردن ارسال پاسخ در تلگرام، متغیر محیطی زیر را تنظیم کنید:

```bash
TELEGRAM_BOT_TOKEN=123456789:abc...
```

پس از استقرار در Vercel، وبهوک ربات را به URL زیر تنظیم کنید:

```
https://agentic-0ed35c59.vercel.app/api/bot
```

### استقرار

برنامه برای استقرار با دستور زیر آماده است:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-0ed35c59
```
